# learngit
利用MYSQL,R,PYTHON的数据库知识以及画图知识，分析2011mozilia_activity——level2中相关数据，从中提取一些有用的信息。其中一些信息注释如下：
Bugzilla状态分类和Bug处理意见（status and Resolution）

1、Bug状态分类（status） 

l 新提交的（new）  

l 已分配（Assigned）  

l 问题未解决的(Reopened)  

l 待返测的(Resolved)  

l 待归档的(Verified)  

l 已关闭的(close)  


2、Bug处理意见（Resolution）  

l 已修改的（Fixed）  

l 不是问题（INVALID）  

l 无法修改（Wontfix）

l 以后版本再解决（Later） 

l 保留（Remind）  

l 重复（Duplicate）  

l 无法重现（Worksforme）
